# Sistema de Adopción de Mascotas 

Este proyecto corresponde a las Tareas 1, 2 y 3 del curso CC5002 - Desarrollo de Aplicaciones WEB.

Para esta entrega se corrigieron todos los errores y se completaron los requisitos pendientes de la **Tarea 1** (prototipo HTML/CSS/JS) y la **Tarea 2** (backend con Flask/MySQL/SQLAlchemy, incluyendo validaciones, subida de archivos y paginación).

Además, se implementaron **todas las funcionalidades de la Tarea 3**, que incluyen:
* **Estadísticas Dinámicas:** Se reemplazaron los gráficos estáticos por gráficos interactivos (líneas, torta y barras) generados con datos reales de la base de datos mediante JavaScript (Fetch) y Highcharts.
* **Sistema de Comentarios Asíncrono:** Se añadió una sección de comentarios en la vista de detalle que permite ver y agregar comentarios sin recargar la página, utilizando Fetch para la comunicación con el backend.

El proyecto utiliza **Python**, **Flask**, **SQLAlchemy**, **MySQL**, **HTML**, **CSS** y **JavaScript**.
